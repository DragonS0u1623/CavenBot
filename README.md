# CavenBot Java
 
CavenBot is a general-purpose discord bot that is written in Java using a mix of JDA and Discord.js. It has moderation, music, react role, and action commands. It can play music from Youtube and Spotify. Future functionality is planned. And I will add a page for seeing the current plan for future updates on the website shortly. 

# Getting Started
You can get started by adding CavenBot to your server [here](https://discord.com/api/oauth2/authorize?client_id=638446270469373972&permissions=1636151717110&scope=bot%20applications.commands)

Help Command  - `m?help`

# Languages
